Gun Match Screw

Wednesday, Sep 17, 2025 | 1 minute read

@ QiGames

Game Type:

A unique, level-based puzzle game combining screw puzzles with gun-themed mechanics.

Objective:

Unlock all the boards, which are shaped like different guns or gun parts, by matching colored screws (represented as bullets).

Gameplay Mechanics:

  • Each board has colored screws that must be matched to unlock panels.
  • The game has two modes where you fill 6-slot bullet magazines.
  • By matching same-colored screws, you fill a magazine, which is then exchanged for a new, empty one.
  • You are also given 5 extra bullet slots for storing unmatched bullets.
